# 📄 맞춤팩 상세화면 PRD (ver. AI 생성 루틴 구조화 기준)

## ✅ 목적
AI가 개인 정보 및 컨텍스트 기반으로 설계한 맞춤팩(주간 루틴 조합)을 사용자가 명확하게 이해하고, 신뢰하며, 섭취 루틴에 자연스럽게 편입할 수 있도록 유도한다.

---

## 맞춤팩 정의
맞춤팩(Custom Pack) 은 1일 1개만 설계됩니다.
이 맞춤팩은 하루의 전체 컨디션을 조절할 수 있도록 섭취 시점별로 나눠서 섭취할 수 있게 구성됩니다.

섭취 시점의 예시는 다음과 같습니다:
아침 - 집중력 강화 / 운동 직후 - 회복 / 저녁 - 수면 or 면역 지원

---

## 🗺 진입 경로
- 홈 루틴 카드 클릭
- 온보딩 완료 후 첫 루틴 제안 시
- 주간 피드백 반영 루틴 생성 시

---

## 🧩 주요 구성 영역 및 설명

| 구성 영역 | 설명 |
|-----------|------|
| **1. 맞춤팩 이름 및 핵심 요약** | 루틴의 주요 목표와 성격을 한눈에 보여줌<br>예: `회복력 + 집중력 케어 루틴` + “강도 높은 훈련 후 피로 회복과 집중력 유지 목적” |
| **2. 설계 배경 요약** | AI가 설계한 배경 요약을 문장 형태로 제공<br>예: “최근 훈련 강도 증가 + 수면 부족 → 회복 저하 위험” |
| **3. 설계에 사용된 사용자 정보** | 루틴 구성의 근거가 된 개인 정보 요약<br>예: `포지션: MF`, `훈련 빈도: 주 5회`, `수면 질 낮음`, `복용 이력: BCAA 1종` |
| **4. 시간대별 섭취 루틴 구성** | 아침 / 훈련 전 / 훈련 후 등 권장 섭취 시점별로 루틴을 분할하여 표시<br>각 시간대별로 섭취 영양소와 목적을 함께 제공 |
| **5. 성능 요약 카드** | 각 루틴 구성의 기대 효과 요약 (카드 UI 구성)<br>예: `근육 회복`, `면역력 유지`, `집중력 증진` |
| **6. 영양소 상세 구성 정보** | 각 시간대별 영양제 이름, 용량, 주요 기능 요약 제공<br>도핑 안정성 표시 포함 |
| **7. 도핑 안전성 및 출처 명시** | 모든 루틴에 “도핑 안전성 검토 완료” 마크 표시 + 제조사/인증기관 정보 표기 |

## 🧠 예시 표현 구조

- **"이번 주는 훈련량이 높고, 수면의 질이 낮았어요. 그래서 회복과 집중력 중심의 조합을 준비했어요."**
- **“훈련 후에는 MSM과 타르트체리를 함께 섭취해 염증 반응을 줄이고 회복을 도와요.”**
- **“이 루틴은 WADA 기준을 기반으로 설계되어 도핑 걱정 없이 섭취할 수 있어요.”**

---

## 📝 기술적 고려사항

- AI 출력 결과는 YAML/JSON 구조를 기반으로 설계되며, 구조화된 루틴 정보를 UI 컴포넌트에 매핑 가능해야 함
- 각 루틴 블록은 시간대 기준으로 반복 가능한 구조로 설계
- 각 성분의 설명은 사전 구축된 DB 혹은 OpenAPI 기반 설명 리치 제공
