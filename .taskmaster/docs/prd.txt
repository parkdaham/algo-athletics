# 🏋️ 알고케어 운동선수 MVP - 루틴 섭취 화면 PRD

## 📌 목적
- 운동선수가 아침 루틴 한 번으로 기본팩 + 부스터를 섭취
- 설계된 루틴의 이유를 이해하고 루틴을 실천에 옮기도록 유도

## 📲 사용자 흐름
1. 홈 진입
2. 루틴 카드 확인
3. 카드 클릭 → 루틴 상세 확인
4. 디바이스로 '지금 먹기' → 루틴 토출

## 🧩 핵심 기능 목록

| 기능     | 설명 |
| -------- | ---- |
| 루틴 요약 카드 | 오늘 섭취 루틴 표시 (기본팩 + 부스터 포함) |
| 루틴 상세 보기 | 섭취 타이밍 별 구성 + 설계 이유 확인 가능 |
| 섭취 실행 버튼 | '지금 먹기' → 디바이스로 전체 토출 |
| 섭취 상태 UI | 일부/전체 섭취 여부에 따른 상태 표시 |
| 주간 루틴 보정 | 주 1회 피드백 기반 자동 설계 업데이트 |
| 추가 부스터 제공 | 루틴 외 상황 대응 부스터는 별도 UI로 제공 |

## ⚠️ 예외 처리
- 디바이스 미연결: "디바이스 연결 필요" 메시지
- 루틴 구성 실패: 기본팩만으로 fallback 루틴 구성

## 💬 예시 문구
- "오늘의 루틴이 준비되었어요. 지금 챙겨볼까요?"
- "격한 훈련 후 회복 중심 루틴을 설계했어요."
